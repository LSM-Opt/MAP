# 🔍 MAP: Model Analysis & Planning Engine

MAP(Model Analysis & Planning Engine)는  
초거대 AI 모델의 구조와 연산 특성을 분석하고,  
이종 AI 반도체 환경에서 최적으로 실행되기 위한 **코드 변환 계획을 수립하는 모듈**입니다.

MAP는 LSM-Opt 파이프라인의 **첫 번째 단계**로서,  
이후 연산 그래프 최적화(GOP), 코드 생성(HCG)의 방향을 결정합니다.

## ✨ Key Responsibilities
- 초거대 AI 모델 구조 분석
- 프레임워크 독립적 모델 특성 추출
- 하드웨어 제약 기반 변환 전략 수립
